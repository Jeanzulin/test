常見問題

Q1/ 不是e.Fingo會員可以參加【e.Fingo感謝季】嗎？

• 此活動為行動銀行App限定活動，只要成功登入玉山行動銀行App後即可參加遊戲，即使尚未成為e.Fingo會員還是可以參加活動唷。
• 此外，您需確認手機時間設定已選擇「自動設定」及玉山行動銀行App iOS版本為 4.3.8 或 Android版本 4.4.0 (含)以上才可以順利參加活動唷。

Q2/ 我需要另外進行活動登錄才能領獎嗎？

不用喔！本行將於活動結束後依據您是否符合兌獎資格陸續寄發贈品。

Q3/ 獎項的認列方式為何？是否可重複獲獎？

您須於2021/7/15前成為e.Fingo會員才可認列活動期間所獲得的【限定e.Fingo會員】獎項唷。 本活動獎項為隨機獲得，若您重複獲獎且符合兌領資格皆可領取

Q4/ 我要如何領取獎項？什麼時候會收到？

• AirPods Pro：預計於2021/8/15前郵寄至您留存於本行之通訊地址。（如需確認或調整通訊地址，請登入「個人網路銀行 / 行動銀行」→「個人設定與服務 / 個人服務」→「基本資料設定」進行設定。）
• 全家購物金200元/威秀影城團體優待票：預計於2021/8/15前以簡訊寄送至您留存於本行之行動電話。（如需確認或調整電話號碼，請登入「個人網路銀行 / 行動銀行」→「個人設定與服務 / 個人服務」→「基本資料設定」進行設定。）
• e.Fingo點數：預計於2021/8/15起可至玉山行動銀行App e.Fingo會員專區點數專區中查詢獲得點數。
• 匯率優惠序號/數位通路基金新申購手續費優惠 3 折/e.Fingo《熊朋友邀請賽》活動抽獎機會：預計於2021/8/1起可登入玉山行動銀行App「我的優惠」專區領取優惠序號及查詢獎項資訊。若您獲得數位通路基金新申購手續費3折優惠，須於2021/7/23前成為本行有效存戶，且必須完成信託總約簽署，始得享有限定期間內基金新申購手續費優惠唷。

Q5/ 各獎項有使用期限嗎？

• 全家購物金200元：使用期限至2021/12/31。
• 威秀影城團體優待票：優惠效期為2021/11/30，過期每張酌收30元手續費，詳細使用說明與規範以威秀影城公告為主。
• e.Fingo點數：使用期限為2022/12/31，有關點數使用方式及會員權益請參照e.Fingo會員權益約定條款。
• 匯率優惠序號：優惠適用期間為2021/8/1 09:00至2021/8/31 23:00止 (使用優惠序號之預約交易付款日期亦須於使用期間內)。
• 數位通路基金新申購手續費優惠 3 折：優惠適用期間為2021/8/1 00:00至2021/9/15 23:00止。
• e.Fingo《熊朋友邀請賽》活動抽獎機會：將累計於《熊朋友邀請賽》7月抽獎機會次數，次月起將重新統計。

Q6/ 如果活動期間不小心刪除玉山行動銀行App，顧客所累積的獲獎紀錄會被取消嗎？

請顧客重新下載玉山行動銀行App，登入後前往獲獎紀錄頁面即可查看顧客於活動期間累積的所有獲獎紀錄。
